# jjvm-2026
proyecto-software-1
El proyecto se basa en una pagina de buscador de compras, la idea principal del proyecto es facilitar la busqueda de precios para las personas al momento decidir donde comprar sus articulos, la pagina lleva como parte de seguridad un login para identificar el ususario con su respectiva contraseña, al ingresar al sistema el usuario entrara en la pagina principal donde saldran sugerencias de articulos diferentes con sus respetivos precios, ademas saldran los productos disponibles, oferta y la cantidad de supermercados, el usuario puede seleccionar que articulo general esta buscando al el articulo aparecera con sus distintos precios por supermercado y indicando si existe stock dentro del supermercado
